HackCambridge

My team and I built a webapp frontend that displays a user's personal "investment tree" linked with the user's stock portfolio. A Python backend hosted on Microsoft Azure tracks the fund composition in real-time and webscrapes environmental, social and corporate governance (ESG) risk scores from Yahoo Finance. A weighted average and other comparisons are computed and made available to the webapp through a Flask-API.

Built with: azure, beautiful-soup, css, flask, html, javascript, json, numpy, pandas, python, regex Won 2nd place in the Microsoft Hack For Good Challenge
